# Student Salary Prediction Using Random Forest Regression

## Overview

This project predicts student salary packages using academic, technical, and professional development indicators.

The model was developed using Random Forest Regression and trained on approximately 20,000 student records.

## Objective

To predict Salary (LPA) using factors such as:

* CGPA
* Internships
* AI/ML Skill Level
* DSA Problems Solved
* LeetCode Rating
* GitHub Contributions
* Communication Skills
* Resume Score
* Mock Interview Score

## Tools Used

* Python
* Google Colab
* Pandas
* Scikit-Learn
* Random Forest Regression

## Methodology

1. Data Exploration
2. Data Preprocessing
3. One-Hot Encoding
4. Train-Test Split
5. Random Forest Model Training
6. Model Evaluation
7. Feature Importance Analysis

## Results

* MAE: 1.64 LPA
* RMSE: 2.06 LPA
* R² Score: 0.32

## Key Insights

The most influential predictors of salary were:

1. CGPA
2. AI/ML Skill Level
3. Internships
4. LeetCode Rating
5. DSA Problems Solved
